GTech Kornell

A fully libre-free Cornell-style note-taking application

Create new Notebook Open existing Notebook

A notebook is a series of Kornells and raw text pages, forming a complete series of written knowledge about a particular subject. You can use it for your learning in general.

A notebook is simply a directory full of .md and .kornell files, so you can easily read and change data in each file. It is, though, more convenient inside GTKornell.

A single Kornell is a Cornell-organized note file that you can open in GTKornell.
